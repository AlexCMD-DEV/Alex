Переменные — это контейнеры для хранения значений данных, таких как числа и символы.

В языке C существуют различные типы переменных (определяемые различными ключевыми словами), например:

int- хранит целые числа без десятичных знаков, например 123или-123
float- хранит числа с плавающей точкой, с десятичными знаками, например 19.99или-19.99
char- сохраняет отдельные символы, такие как 'a'или 'B'. Символы заключаются в одинарные кавычки

Объявление (создание) переменных
Чтобы создать переменную, укажите тип и присвойте ей значение :

Синтаксис
type variableName = value;
Где type — один из типов C (например, int), а variableName — имя переменной (например, x или myName ). Знак равенства используется для присвоения значения переменной.

Итак, чтобы создать переменную, которая должна хранить число , рассмотрим следующий пример:

Пример
Создайте переменную с именем myNum типа и присвойте ей intзначение 15 :

int myNum = 15;
Вы также можете объявить переменную, не присваивая ей значение, и присвоить значение позже:

Пример
// Declare a variable
int myNum;

// Assign a value to the variable
myNum = 15;

Выходные переменные
Из главы «Вывод» вы узнали , что можно выводить значения/печатать текст с помощью printf()функции:

Пример
printf("Hello World!");

Во многих других языках программирования (например, Python , Java и C++ ) вы обычно используете функцию печати для отображения значения переменной. Однако в C это невозможно:

Пример
int myNum = 15;
printf(myNum);  // Nothing happens
