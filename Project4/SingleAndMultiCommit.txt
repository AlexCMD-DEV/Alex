Комментарии на языке C
Комментарии можно использовать для пояснения кода и для того, чтобы сделать его более читаемым. Также их можно использовать для предотвращения выполнения при тестировании альтернативного кода.

Комментарии могут быть однострочными или многострочными .

Однострочные комментарии
Однострочные комментарии начинаются с двух косых черт ( //).

Любой текст между //строкой и ее концом игнорируется компилятором (не будет выполнен).

В этом примере используется однострочный комментарий перед строкой кода:

Пример
// This is a comment
printf("Hello World!");

C Многострочные комментарии
Многострочные комментарии начинаются с /*и заканчиваются на */.

Любой текст между /*и */будет проигнорирован компилятором:

Пример
/* The code below will print the words Hello World!
to the screen, and it is amazing */
printf("Hello World!");

Однострочные или многострочные комментарии?
Вам решать, что вы хотите использовать. Обычно мы используем //для коротких комментариев и /* */для более длинных.

Полезно знать: до версии C99 (выпущенной в 1999 году) в языке C можно было использовать только многострочные комментарии.
